# Real-Time Weather Data Pipeline

## Description: 
Automated pipeline that collects, cleans, and stores live weather data

## Tech Focus: 
API integration • ETL • Scheduling • Visualization

## Features:
- Pull live weather data from a OpenWeatherMap API
- Clean it using pandas, standardize timestamps, and handle missing data
- Load it into a PostgreSQL or BigQuery table
- Automate the process daily using Airflow or cron
- Present a dashboard of temperature trends and anomalies
- Optional: Alerting
